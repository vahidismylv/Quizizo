# Quizizo

UIKit-based quiz game app built as a final project.

## Features
- Sign in with Google and Apple
- Token-based backend authorization

- Custom networking layer (URLSession + Codable)
- Secure token storage (Keychain)
- Modular UIKit UI with reusable container views
- Timed quiz gameplay (timer, answers, score)
- Record / stats screen

## Tech Stack
Swift · UIKit · GoogleSignIn · AuthenticationServices · URLSession · Codable · Keychain

Login · Home · Game · Record
